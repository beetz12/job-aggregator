# Company Evaluation Scoring Rubric

This document provides detailed scoring criteria for each evaluation category.

---

## Category 1: Compensation (20 Points)

### Base Salary (10 points)

| Score | Criteria |
|-------|----------|
| 10 | $230K+ base (exceeds target) |
| 8 | $200K-$229K (meets target) |
| 6 | $180K-$199K (meets floor) |
| 3 | $160K-$179K (below floor but negotiable) |
| 0 | Below $160K (disqualifying) |

### Equity (5 points)

| Score | Criteria |
|-------|----------|
| 5 | Strong equity (0.25%+ early stage, competitive RSUs late stage) |
| 4 | Good equity (stage-appropriate) |
| 3 | Standard equity |
| 1 | Minimal equity |
| 0 | No equity offered |

### Benefits (5 points)

| Score | Criteria |
|-------|----------|
| 5 | Exceptional: All must-haves + strong preferences |
| 4 | Good: All must-haves + some preferences |
| 3 | Acceptable: Meets must-haves |
| 2 | Below average: Missing some must-haves |
| 0 | Poor: Missing multiple must-haves |

**Must-Have Benefits:**
- Health insurance (medical, dental, vision) covering family
- 401(k) with company match (4%+)
- Parental leave (12+ weeks)
- Remote work support
- Professional development budget

## Category 2: Culture & Leadership (25 Points)

### Glassdoor Rating (8 points)

| Score | Criteria |
|-------|----------|
| 8 | 4.5+ rating with 100+ reviews |
| 6 | 4.0-4.4 rating |
| 4 | 3.5-3.9 rating |
| 2 | 3.0-3.4 rating |
| 0 | Below 3.0 or no reviews |

### CEO/Leadership Approval (5 points)

| Score | Criteria |
|-------|----------|
| 5 | 90%+ CEO approval |
| 4 | 80-89% approval |
| 3 | 70-79% approval |
| 1 | 50-69% approval |
| 0 | Below 50% approval |

### Servant Leadership Evidence (7 points)

| Score | Criteria |
|-------|----------|
| 7 | Strong evidence: Public credit sharing, psychological safety |
| 5 | Moderate evidence: Positive leadership mentions |
| 3 | Neutral: No strong evidence |
| 1 | Concerning: Some negative leadership reviews |
| 0 | Red flag: Blame culture, toxic leadership |

**What to Look For:**
- Leaders publicly crediting teams
- Leaders taking responsibility for failures
- Employee reviews mentioning psychological safety
- Low regrettable turnover

### Awards & Recognition (5 points)

| Score | Criteria |
|-------|----------|
| 5 | Multiple "Best Places to Work" awards |
| 4 | One major award in last 2 years |
| 3 | Industry-specific recognition |
| 1 | No awards but positive reputation |
| 0 | Negative press or reputation issues |

## Category 3: Family-Friendliness (20 Points)

| Score | Criteria |
|-------|----------|

| 6 | 16-19 weeks |
| 5 | 12-15 weeks (minimum) |
| 3 | 8-11 weeks (below minimum) |

| Score | Criteria |
|-------|----------|

| 3 | Standard flexibility |
| 1 | Limited flexibility |
| 0 | No flexibility (fixed schedule) |

| Score | Criteria |
|-------|----------|
| 4 | Fully remote, no office requirement |
| 3 | Remote-first with optional office |
| 2 | Hybrid (1-2 days/week) |
| 1 | Hybrid (3+ days/week) |
| 0 | On-site required (disqualifying) |

| Score | Criteria |
|-------|----------|
| 3 | Parents in leadership, childcare benefits, family ERGs |
| 2 | Some family-friendly indicators |
| 1 | Neutral |
| 0 | Work-life balance issues in reviews |

## Category 4: Technical Fit (15 Points)

### Tech Stack Alignment (6 points)

| Score | Criteria |
|-------|----------|
| 6 | Perfect: React/Next.js + Node.js/Python + AI/LLMs |
| 5 | Strong: 3+ core technologies |
| 4 | Good: 2 core technologies |
| 3 | Partial: 1 core technology |
| 1 | Minimal: Related technologies |
| 0 | No match: Completely different stack |

**Core Technologies:**
- Frontend: React, Next.js, TypeScript
- Backend: Node.js, Python (FastAPI/Django/Flask)
- AI/ML: LLMs (Claude, GPT, Gemini), RAG, vector databases
- Infrastructure: AWS, Docker, Vercel, Supabase

### Technical Challenge Level (5 points)

| Score | Criteria |
|-------|----------|
| 5 | Complex: Scale, AI/ML, greenfield |
| 4 | Engaging: Interesting problems |
| 3 | Standard: Typical SaaS |
| 1 | Basic: Maintenance or CRUD |
| 0 | Legacy with no modernization path |

### Learning & Growth (4 points)

| Score | Criteria |
|-------|----------|
| 4 | Strong L&D: $2K+ budget, conferences, mentorship |
| 3 | Good L&D: Budget exists |
| 2 | Basic L&D: Some resources |
| 1 | Limited: "Learn on your own" |
| 0 | None |

## Category 5: Industry & Mission (10 Points)

### Industry Category (6 points)

| Score | Category |
|-------|----------|
| 6 | Tier 1: AI/ML platforms, Developer tools, B2B SaaS, E-commerce |
| 4 | Tier 2: Fintech (modern), Healthcare (innovative), Climate tech |
| 3 | Tier 3: Consumer tech, Education, Real estate |
| 2 | Proceed with Caution: Healthcare (traditional), Big Tech |
| 0 | Excluded: Banks, Government, Defense, Insurance, Gambling, Tobacco, Crypto |

### Mission Alignment (4 points)

| Score | Criteria |
|-------|----------|
| 4 | Exciting: Genuinely care about the problem |
| 3 | Interesting: Would enjoy talking about it |
| 2 | Neutral: Don't mind |
| 1 | Tolerable: Can do it |
| 0 | Misaligned: Don't believe in mission |

### Career Ladder (4 points)

| Score | Criteria |
|-------|----------|
| 4 | Clear IC track: Senior -> Staff -> Principal -> Distinguished |
| 3 | Good IC path |
| 2 | Limited: Growth unclear |
| 1 | Flat: Little room |

| Score | Criteria |
|-------|----------|
| 3 | Growing: Strong growth, recent funding |
| 2 | Stable: Profitable or well-funded |
| 1 | Uncertain: Mixed signals |
| 0 | Declining: Layoffs, revenue decline |

### 5-10 Year Fit (3 points)

| Score | Criteria |
|-------|----------|
| 3 | Strong: Could see 5-10 years |
| 2 | Good: 3-5 year fit |
| 1 | Short-term: 1-2 year opportunity |
| 0 | Poor: Would leave quickly |

## Score Interpretation

| Range | Recommendation |
|-------|----------------|
| 85-100 | STRONG YES |
| 70-84 | YES |
| 55-69 | MAYBE |
| 40-54 | WEAK |
| Below 40 | PASS |

## Category Minimums

Even with good total score, recommend PASS if any critical minimum not met:

| Category | Minimum | Rationale |
|----------|---------|-----------|
| Compensation | 6/20 | Must meet salary floor |
| Culture & Leadership | 10/25 | Culture non-negotiable |
| Family-Friendliness | 8/20 | Parent priority |
| Technical Fit | 6/15 | Must be engaging |
| Industry & Mission | 4/10 | No excluded industries |
| Long-Term Potential | 3/10 | Some growth needed |
